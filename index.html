Gilliginsisland Repository

Welcome to Gilliginsisland's ubuntu repository!

To start downloading software from this repository, copy the code snippet bellow to add the repository for your distribution.

wget -qO - http://apt.broochian.com/archive.key | sudo apt-key add -
echo 'deb http://apt.broochian.com ubuntu main' | sudo tee /etc/apt/sources.list.d/gilliginsisland.list
apt update

That's it! You can now install any packages from this repository.
