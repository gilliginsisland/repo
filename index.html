Gilliginsisland Repository

Welcome to Gilliginsisland's ubuntu repository!

Import the apt signing key to your keyring

wget -qO - http://apt.broochian.com/archive.key | sudo apt-key add - echo 'deb [allow-insecure=yes] http://apt.broochian.com ubuntu main' | sudo tee /etc/apt/sources.list.d/gilliginsisland.list apt update

That's it! You can now install any packages from this repository.
